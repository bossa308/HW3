# ⚡ ระบบจัดการโหลดไฟฟ้า (hw3)

Web App คำนวณ **กำลังไฟ · พลังงาน · ค่าไฟ** ของอุปกรณ์ไฟฟ้าในบ้าน
สร้างด้วย **Next.js + TypeScript + Tailwind CSS** และ **Deploy บน Vercel**

> ต่อยอดจาก Hw4 (Google Apps Script) มาเป็น Web App สมัยใหม่ที่รันบนอินเทอร์เน็ตได้จริง

---

## 🔬 สูตรวิศวกรรมไฟฟ้าที่ใช้
| ปริมาณ | สูตร |
|--------|------|
| กำลังไฟฟ้า | `P = V × I` |
| กระแสจากกำลังไฟ | `I = P / V` |
| กฎของโอห์ม | `R = V / I` |
| พลังงาน/เดือน | `kWh = P(W) × ชม./วัน × 30 ÷ 1000` |
| ค่าไฟ | `ค่าไฟ = kWh × อัตรา(บาท/หน่วย)` |

---

## ✨ ฟีเจอร์
- 📐 **เครื่องคำนวณสด** — โชว์ P, I, R, พลังงาน, ค่าไฟ ทันทีขณะพิมพ์
- ➕ เพิ่มอุปกรณ์ได้ทั้งแบบ **กำลังไฟ (W)** หรือ **กระแส (A)**
- 📊 การ์ดสรุปยอด + กราฟพลังงานแยกตามอุปกรณ์
- 📋 ตารางอุปกรณ์ พร้อม **ค้นหา** และ **ลบ**
- 💾 บันทึกข้อมูลอัตโนมัติใน `localStorage` (ปิดเปิดเบราว์เซอร์ข้อมูลไม่หาย)

---

## 🗂️ โครงสร้างโปรเจกต์ (แต่ละไฟล์ทำอะไร)

```
src/
├── app/
│   ├── layout.tsx      # โครงหน้าเว็บ + ฟอนต์ไทย + <title>
│   ├── page.tsx        # หน้าหลัก = "สมองกลาง" ถือ state ทั้งหมด
│   └── globals.css     # ตั้งค่า Tailwind + ฟอนต์ + สีพื้นหลัง
├── lib/
│   ├── calc.ts         # สูตรคำนวณไฟฟ้า (ฟังก์ชันล้วน ไม่มี UI)
│   └── storage.ts      # อ่าน/เขียน localStorage + ข้อมูลตัวอย่าง
└── components/
    ├── SummaryCards.tsx    # การ์ดสรุป 4 ใบด้านบน
    ├── LiveCalculator.tsx  # ฟอร์มกรอก + คำนวณสด + ปุ่มเพิ่ม
    ├── EnergyChart.tsx     # กราฟแท่งพลังงาน
    └── DeviceTable.tsx     # ตารางอุปกรณ์ + ค้นหา + ลบ
```

**แนวคิดการไหลของข้อมูล:** `page.tsx` เก็บ `devices` ไว้ที่เดียว → ส่งลงไปให้ลูก ๆ ผ่าน props →
เวลาผู้ใช้กดเพิ่ม/ลบ ลูกจะเรียก callback กลับขึ้นมาแก้ที่ `page.tsx` → ทุกส่วนอัปเดตพร้อมกัน

## 🖥️ รันบนเครื่องตัวเอง (Local)

```bash
npm install      # ติดตั้งครั้งแรกครั้งเดียว
npm run dev      # เปิด http://localhost:3000
```

คำสั่งอื่น: `npm run build` (สร้างเวอร์ชัน production) · `npm start` (รัน production)

## 🚀 Deploy ขึ้น Vercel

### วิธี A — ผ่าน GitHub (แนะนำ ใช้ง่ายสุด)
1. สร้าง repo ใหม่บน **GitHub** แล้ว push โค้ดขึ้นไป
   ```bash
   git add -A
   git commit -m "hw3: electrical load web app"
   git branch -M main
   git remote add origin https://github.com/<ชื่อคุณ>/hw3.git
   git push -u origin main
   ```
2. เข้า **https://vercel.com** → ล็อกอินด้วย GitHub → กด **Add New… → Project**
3. เลือก repo `hw3` → Vercel ตรวจเจอว่าเป็น Next.js เอง → กด **Deploy**
4. รอสักครู่ จะได้ลิงก์ใช้งานจริง เช่น `https://hw3-xxxx.vercel.app` ✅
   *(ทุกครั้งที่ push ขึ้น GitHub ใหม่ Vercel จะ deploy ให้อัตโนมัติ)*

### วิธี B — ผ่าน Vercel CLI (ไม่ต้องใช้ GitHub)
```bash
npm i -g vercel     # ติดตั้ง CLI ครั้งเดียว
vercel              # ครั้งแรกให้ล็อกอิน แล้วตอบ Enter ผ่านได้หมด
vercel --prod       # deploy เวอร์ชันจริง
```

## 📝 หมายเหตุ
- เวอร์ชันนี้เก็บข้อมูลใน **เบราว์เซอร์** (localStorage) จึงไม่ต้องตั้งค่าฐานข้อมูล — เหมาะกับงานเดี่ยว/เดโม
- ถ้าต้องการเก็บข้อมูลรวมศูนย์ (หลายคนเห็นข้อมูลเดียวกัน) ขั้นต่อไปคือเพิ่ม **API Route + ฐานข้อมูล**
  (เช่น Vercel Postgres) แล้วแก้แค่ 2 ฟังก์ชันใน `src/lib/storage.ts` ให้ยิง API แทน
