> *(ใช้: Claude Code – Claude Opus 4.8 ; รันคำสั่ง `create-next-app`)*

### 3. เขียนสูตรคำนวณ + ระบบเก็บข้อมูล
> "เขียนไฟล์สูตรไฟฟ้าเป็นฟังก์ชันแยกจาก UI — P = V×I, I = P/V, R = V/I,
> พลังงาน kWh/เดือน = P×ชม.×30/1000, ค่าไฟ = kWh×อัตรา ;
> และไฟล์จัดการ localStorage พร้อมข้อมูลตัวอย่าง 5 อุปกรณ์"
> *(ใช้: Claude Code – Claude Opus 4.8)*

### 4. สร้างหน้าเว็บ (UI ภาษาไทย)
> "สร้างคอมโพเนนต์: การ์ดสรุปยอด, เครื่องคำนวณสด (โชว์ผลทันทีขณะพิมพ์ + ปุ่มเพิ่ม),
> กราฟแท่งพลังงาน, ตารางอุปกรณ์ที่ค้นหา/ลบได้ แล้วประกอบเข้าหน้าหลัก เป็นภาษาไทยทั้งหมด"
> *(ใช้: Claude Code – Claude Opus 4.8)*

### 5. รันทดสอบ + แก้บั๊ก

> *(ใช้: Claude Code – Claude Opus 4.8)*
